# hrishikeshpatil_bitsom_ba_2511159_part4_tableau_dashboard
Business Problem
The leadership team needed one single dashboard where they could see the overall health of the business — covering sales, profit, customer segments, regional performance, product categories, shipping, discounts, and returns. Instead of checking multiple reports, they wanted a clear view to spot opportunities, risks, and areas that need immediate attention.
Dataset Description
I used the dashboard_sales_data.xlsx file, which contains detailed retail sales records. It includes key information like Order Date, Sales, Profit, Category, Sub-Category, Region, Customer Segment, Ship Mode, Discount, Delivery Days, Return Flag, and Product Name.
Tableau Workbook
The final Tableau packaged workbook is saved as tableau/executive_dashboard.twbx. It contains all the worksheets, calculated fields, the main dashboard, filters, and interactions.
Calculated Fields Created
I created a few important calculated fields to make the analysis more meaningful:
Profit Margin
Cost
Average Order Value
Return Rate
Shipping Delay Bucket (Fast, Normal, Delayed)
These calculations helped bring out clearer business insights across the dashboard.
Dashboard Components
The executive dashboard includes KPI cards for Total Sales, Total Profit, and Profit Margin at the top. The main visuals cover:
Monthly Sales Trend
Regional Performance
Category Profitability
Customer Segment Analysis
Shipping Performance
Discount vs Profit Analysis
Return Analysis
